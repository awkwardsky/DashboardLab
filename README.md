# DashboardLab

DashboardLab is a static dashboard experiment for testing different data presentation patterns:

- KPI cards
- Chart panels
- Region summaries
- Sortable metric tables
- Activity timelines
- System status and alert blocks

## Local Preview

Run this from the project folder:

```bash
python3 -m http.server 8000
```

Then open:

```text
http://localhost:8000
```

## GitHub Pages

This project follows the same deployment style as `ReactionSpeedLab`: a standalone GitHub repository served by GitHub Pages from the `main` branch root.
